# TextileHub - Global B2B Textile Marketplace

A modern B2B marketplace platform connecting textile buyers with verified suppliers worldwide.

## Features

- Browse and search 10,000+ verified textile suppliers
- Direct manufacturer connections
- Secure transactions and quality assurance
- Competitive bulk pricing with MOQ flexibility
- User authentication and authorization
- Responsive design for all devices

## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn-ui
- **Backend**: Supabase
- **Routing**: React Router
- **State Management**: TanStack Query

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd textilehub

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Deployment

Build the project for production:

```sh
npm run build
```

The build output will be in the `dist` directory, ready to be deployed to any static hosting service.
